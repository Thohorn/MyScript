# context.md

## Goal

Build a simple, persistent Todo app component in Vue 3 using TypeScript.

## Tech Stack

- Vue 3
- TypeScript
- Vite

## Architecture

- Single File Component (SFC)
- Composition API
- Todos stored in localStorage
- Scoped CSS for component styles

## Key Rules

- Use Vue 3 Composition API for logic
- TypeScript strict mode for type safety
- Follow Vue 3 best practices (props, emits, reactivity)
- Keep UI and logic modular and clean
- Persist todos in localStorage on every change
- Use clear, typed interfaces for todos
- Minimal, readable code
- Use computed() for derived state
- Use defineProps<{ ... }> for type-safe props
